Yahtzee

Grading Rubric

There are several ways that you can organize your files for this project. I will describe my prefered method in the document below, but feel free to do whatever works best for you.

In IntelliJ, create a new Java project called Yahtzee. We will then make a module for each level. In order to use the class files provided in Level Two and higher, you will need to add the folders I provide for each level as dependencies for that module.

Yahtzee Level I (0-59 points)

  • Create a LevelOne module in your project.
  • Download (right click and Save File As) and place the following files in that project:
    • YahtzeeDie.java
    • YahtzeeDieTestDriver.java
  • Run YahtzeeDie.java and YahtzeeDieTestDriver.java
  • Complete the stub methods in YahtzeeDie class.
  • Use the YahtzeeDieTestDriver.java to test and see that your YahtzeeDie class is working successfully.
  • Before moving on to Level II BE SURE YahtzeeDie.java is fully tested. You should not make any changes to YahtzeeDie.java after this point!

Yahtzee Level II (60-69 points)

  • Create a LevelTwo module in your project.
  • Copy the YahtzeeDie.java file you wrote from your YahzteeLevelOne project.
  • Download (right click and Save File As) and place the following files in that project:
    • YahtzeeDriver.java
    • YahtzeeGame.class
    • YahtzeeScorecard.class
    • YahtzeeSortDice.class
  • The .class files are already compiled classes you will have the opportunity to write in future levels. In this form you can use them, so you have a working program, without being able to see the code that allows them to work.
  • You will need to put them in a folder and add that folder as a dependency to your module.
  • Compile and run your project.
  • Execute YahtzeeDriver.java
  • Complete the main method in YahtzeeDriver.java
  • Before moving on to Level III BE SURE YahtzeeDriver.java is fully tested. You should not make any changes to YahtzeeDriver.java after this point!
    • \

Yahtzee Level III (70-89 points)

  • Create a LevelThree module in your project.
  • Copy the YahtzeeDie.java and YahtzeeDriver.java files you wrote from your YahzteeLevelTwo project
  • Download (right click and Save File As) and place the following files in that project:
    • YahtzeeGame.java
    • YahtzeeScorecard.class
    • YahtzeeSortDice.class
  • Compile and run your project.
  • Execute YahtzeeDriver.java
  • Complete the methods in YahtzeeGame.java
  • Before moving on to Level IV BE SURE YahtzeeGame.java is fully tested. You should not make any changes to YahtzeeGame.java after this point!

Yahtzee Level IV (90-100 points)

  • Create a LevelFour module in your project.
  • Copy the YahtzeeDie.java, YahtzeeDriver.java and YahtzeeGame.java files you wrote from your YahzteeLevelThree project.
  • Download (right click and Save File As) and place the following files in that project:
    • YahtzeeScorecard.java
    • YahtzeeSortDice.class
  • Compile and run your project.
  • Execute YahtzeeDriver.java
  • Complete the methods in YahtzeeScorecard.java
  • Before moving on to Level V BE SURE YahtzeeScorecard.java is fully tested. You should not make any changes to YahtzeeScoreboard.java after this point!

Yahtzee Level V (100-105 points)

  • Create a LevelFive module in your project.
  • Copy the YahtzeeDie.java, YahtzeeDriver.java, YahtzeeGame.java and YahtzeeScorecard.java files you wrote from your YahzteeLevelFour project.
  • Download (right click and Save File As) and place the following files in that project:
    • YahtzeeSortDice.java
  • Compile and run your project.
  • Execute YahtzeeDriver.java
  • Complete the methods in YahtzeeSortDice.java

Once you are done, please consult Canvas for submission instructions.
